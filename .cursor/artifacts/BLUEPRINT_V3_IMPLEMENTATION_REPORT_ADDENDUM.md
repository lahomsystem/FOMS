
## 🛠️ 7. 추가 개선 사항 (2026-02-07 22:30 업데이트)

### 7.1 도면 전달 알림 시스템
**구현 현황**:
- `Notification` 모델 추가: 담당자(라홈/하우드/영업사원) 및 팀(CS/HAUDD/SALES) 구분 저장
- **자동 알림 생성**: 도면 전달(`api_order_transfer_drawing`) 시 담당자에 따라 타겟 설정
  - 담당자명 '라홈' 포함 → 라홈팀(CS) 알림
  - 담당자명 '하우드' 포함 → 하우드팀(HAUDD) 알림
  - 그 외 → 해당 영업사원(SALES) 알림
- **대시보드 UI**:
  - 통합/생산/시공 대시보드 헤더에 알림 벨 및 배지 추가
  - 알림 패널 구현 (목록 조회, 읽음 처리, 클릭 시 주문 이동)

### 7.2 UX 개선
- **저장 후 리다이렉트**:
  - 신규 주문 저장 시 → 전체 목록(`/erp`)으로 이동
  - 기존 주문 수정 시 → 해당 주문 상세(`/edit/{id}`)로 이동
  - 저장 완료 메시지 후 0.8초 딜레이 적용으로 사용자 인지 강화
