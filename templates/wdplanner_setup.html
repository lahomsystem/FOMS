WDPLANNER 설정 안내

WDPlanner를 사용하려면 먼저 빌드 파일을 생성해야 합니다.

1단계: WDPlanner 디렉토리로 이동

cd "Add In Program\WDPlanner"

2단계: 의존성 설치 (처음 한 번만)

npm install

3단계: 웹 빌드 실행

npm run build

이 명령은 dist 폴더에 웹 빌드 파일을 생성합니다.

4단계: 빌드 파일 복사

빌드가 완료되면 dist 폴더의 모든 파일을 다음 위치로 복사하세요:

static\wdplanner\

또는 다음 명령을 사용할 수 있습니다 (PowerShell):

xcopy /E /I /Y "Add In Program\WDPlanner\dist\*" "static\wdplanner\"

5단계: 확인

다음 파일이 존재하는지 확인하세요:

static\wdplanner\index.html

파일이 준비되면 이 페이지를 새로고침하거나 WDPLANNER 메뉴를 다시 클릭하세요.

주의사항:
  • 빌드 파일이 없으면 WDPlanner를 사용할 수 없습니다.
  • WDPlanner 소스 코드를 수정한 후에는 다시 빌드해야 합니다.
  • 빌드 파일은 static/wdplanner 폴더에 있어야 합니다.
